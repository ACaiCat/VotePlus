# VotePlus (更好的投票)  

----------
## 安装方法:

**1.将插件安装在ServerPlugins/  
2.重启服务器**  


--------
## 使用方法:  
**/vote kick <玩家> 投票踢出玩家  
/vote ban <玩家> 投票封禁玩家  
/vote mute <玩家> 投票禁言玩家  
/vote clearboss <BOSS名称> 投票清理BOSS(不掉落)  
/vote clearevent 投票清理地图中所有事件  
/vote day 投票将时间改为白天  
/vote night 将时间改为黑夜  
/vote rain 投票下雨或停止下雨  
/vote topic <主题> 自由主题投票  
/vote clearkick <玩家> 将玩家从踢出状态移除   
/vote kicklist 查看处于踢出状态的玩家  
/agree 同意投票  
/disagree 反对投票**  
## 权限:  vote.use  
## 配置文件: tshock/VotePlus.json
